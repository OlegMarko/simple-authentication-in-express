## Install app

`git clone git@github.com:OlegMarko/simple-authentication-in-express.git`

`cd simple-authentication-in-express`

`npm i`

`cp .env.example .env`

- put credentials from your [Okta](https://developer.okta.com/) account to `.env`

`npm start`

```
    requirements
    
    - node v8.*
    - npm 3.*
```
